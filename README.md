# Post-Quantum Cryptography Accelerator on FPGA
This undergraduate senior project focuses on accelerating key operations in Kyber, a lattice-based post-quantum cryptographic scheme.

Goal : Implement and optimize cryptographic modules using FPGA hardware to achieve faster and more energy-efficient computation 

## Operations
- Polynomial Generation
- Encapculation
- Decapsulation

## Hardware Modules
- NTT (Number theoretic transform)
- SHAKE128
- Decode
- Encode
